# Requirements: Gentle Games

**Defined:** 2026-03-03
**Core Value:** A safe, predictable, and stress-free digital space where children can play, create, and explore at their own pace.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Learning Games

- [ ] **LEARN-01**: NumberPicnic - User can play number learning game (1-10)
- [ ] **LEARN-02**: NumberPicnic - Numbers displayed with visual representations
- [ ] **LEARN-03**: LetterLantern - User can play letter learning game (A-Z)
- [ ] **LEARN-04**: LetterLantern - Letters displayed with visual representations
- [ ] **LEARN-05**: StarPath - User can collect stars through simple interactions

### Game Polish

- [ ] **POLISH-01**: All games work with animations disabled
- [ ] **POLISH-02**: All games work with sound disabled
- [ ] **POLISH-03**: All games have consistent UI patterns

### Accessibility

- [ ] **ACCESS-01**: Touch targets minimum 48dp
- [ ] **ACCESS-02**: Supports light/dark mode
- [ ] **ACCESS-03**: Supports reduced motion preference

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Parent Features

- **PARENT-01**: Parent dashboard showing play time
- **PARENT-02**: Activity history per session

### Content

- **CONTENT-01**: Additional drawing tools (colors, brushes)
- **CONTENT-02**: Additional breathing exercises

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Multiplayer | Not aligned with calm, private play |
| User accounts | Adds complexity, privacy concerns |
| Cloud sync | Contradicts offline-first principle |
| Social sharing | Not aligned with core value |
| Timed challenges | Creates pressure |
| Leaderboards | Creates competition |
