Gentle Games does not collect, store, or share any personal data. Ever.

This Privacy Policy describes our data practices for the Gentle Games mobile application (available on iOS and Android) and this website. We are committed to protecting the privacy of all users, and particularly the children who use this application.

1. Information We Do Not Collect

Gentle Games does not collect any of the following:

  • Names, email addresses, or any other contact information
  • Device identifiers (IDFA, GAID, or similar advertising IDs)
  • IP addresses or location data
  • Usage analytics or crash reports sent to third-party services
  • Photos, microphone audio, or camera data
  • Payment information
  • Any information about children

2. Data Stored Locally on Your Device

The app saves a small set of preferences to your device using AsyncStorage. This data never leaves your device and is used solely to remember your settings between sessions:

  • Sound on/off and volume level
  • Animation on/off preference
  • Last selected pair count for Memory Snap

You can clear this data at any time by uninstalling the application.

3. No Network Requests

During normal use, Gentle Games makes no network requests. All game content and assets are bundled with the application. There are no remote servers, no APIs called, and no data transmitted over the internet.

4. No Third-Party SDKs That Track Users

Gentle Games does not integrate any advertising networks, analytics platforms, social media SDKs, or crash-reporting services that collect user data. The only third-party libraries used are open-source UI and audio frameworks (React Native, Expo, React Navigation, Expo AV, React Native SVG, and AsyncStorage) that operate entirely on-device.

5. Children's Privacy (COPPA / GDPR-K)

Gentle Games is designed for children ages 4‚Äì10. Because we collect no personal information whatsoever, the app fully complies with the Children's Online Privacy Protection Act (COPPA) and equivalent regulations. No parental consent is required because no data is collected from users of any age.

6. No Ads or In-App Purchases

Gentle Games contains no advertisements and no in-app purchases. There are no paid features, no premium tiers, and no subscription models.

7. Open Source

The complete source code for Gentle Games is publicly available on GitHub under the GNU General Public License v3. You can inspect exactly what the app does at any time.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Effective date" at the top of this page and in the repository commit history. Because we collect no data, changes will typically only be made to improve clarity.

9. Contact

If you have any questions about this Privacy Policy, please open an issue on the GitHub issue tracker.
